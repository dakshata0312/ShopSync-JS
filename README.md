# Simple E-Commerce 🛒

A lightweight, interactive shopping cart application. This project simulates the core functionality of an online store, allowing users to browse products and manage their selections seamlessly.

## 🚀 Features
* **Dynamic Product Listing:** Products are generated and rendered using JavaScript.
* **Live Cart Management:** Add items to the cart with real-time UI updates.
* **Total Calculation:** Automatically calculates the total cost of all items in the cart.
* **Conditional UI:** Shows "Your cart is empty" or the Checkout button based on the cart's state.
* **Modern UI:** Built with a sleek dark-mode aesthetic.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3
* **Scripting:** JavaScript (ES6+)

## 📖 What I Learned
Through this project, I strengthened my understanding of:
1. **Array Methods:** Using `.forEach()` to render products and `.reduce()` or `.push()` for cart logic.
2. **State Syncing:** Ensuring the "Cart" display matches the internal data array at all times.
3. **Event Delegation:** Handling clicks on dynamically created "Add to Cart" buttons.
4. **Logic Toggling:** Using CSS classes to show/hide the checkout summary based on cart length.
